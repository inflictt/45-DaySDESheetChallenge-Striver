## 📌 About

This repository contains my complete solutions for the **45-Day Striver SDE Sheet Challenge** by **TakeUForward**.

The goal of this challenge was to strengthen my understanding of **Data Structures & Algorithms**, improve problem-solving skills, and build consistency while preparing for Software Engineering interviews.

Each day consists of carefully selected interview questions covering the most important DSA topics frequently asked in top product-based companies.
